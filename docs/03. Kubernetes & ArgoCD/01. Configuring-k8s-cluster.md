## Configuring Kubernetes cluster on Ubuntu

```sh
curl -LO https://storage.googleapis.com/minikube/releases/latest/minikube_latest_amd64.deb
sudo dpkg -i minikube_latest_amd64.deb
```

If you are using any other machine, refer [docs](https://minikube.sigs.k8s.io/docs/start/)
